Question : Given array of N elements, rotate the array in clock wise direction by k times.

Hint : for every rotation last Elemnt comes to first place

If k = length of the array , result will be the Original array

Program : 
Brute force solution : 
a = [1,2,3,4,5,6,7,8,9]
k = int(input()) //input from the user
for i in range(k):
    ele = a.pop()
    a.append(ele)
print(a) 

Optimized  solution : 
a = [1,2,3,4,5,6,7,8,9]
k = int(input()) //input from the user
k = k % len(a)   // if k is greater than length of the array
print(a[len(a)-k : ] + a[:len(a)-k])
Output :

if k = 1 : [9, 1, 2, 3, 4, 5, 6,7,8]

if k = 2 : [8,9, 1, 2, 3, 4, 5, 6,7]

if k = 3 : [7,8,9, 1, 2, 3, 4, 5, 6]
