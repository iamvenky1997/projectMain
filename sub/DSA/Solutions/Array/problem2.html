Question : Given n array elements, check if there exists a pair (i,j) such that a[i] + a[j] = k, Where (i != j)
Brute-Force Solution : 

a = [4,-2,1,3,0,-4,6,8]
k = 10
for i in range(len(a)-1):
    for j in range(i+1,len(a)):
        if a[i] + a[j] == k:
            print(f"index : {i},{j}")
        
else:
    print("No pair found")
