Question :Given n element in an array, count the number of elements having at least 1 element greater than itself
Algorithm :

If we find the maximum element in an array. Then we can say all the other elements are less than the maximum element. By this observation we can easily find out the solution

  1. Find the maximum element of given array
  2. Find the count of maximum in the array
  3. Subtract total number of elements in an array with count of maximum number in given array
solution 1 : 
a = [2,3,53,34,3,4,5,6,7,8,4,53,3,53]
max = a[0]
count = 1
for i in a:
    if max < i:
        max = i
        count = 1
    elif max == i:
        count += 1

res = len(a) - count
print(f"{res} having at least 1 element greater than itself in the given arrar.")
output :

11 having at least 1 element greater than itself in the given arrar.
