Search an element In an array by using Binary search

This is also the most typical question asked for many freshers to test there basic knowledge. There are many ways to solve this problem. In this article i am going to show you by using binary search. we will use python programming for solving this question. you may use same logic to solve in other programming language. The logic is same for every programm, but syntax may vary.

Steps to solve the problem

  1. To perform Binary search, the array should be in the form of sorted like this [2,3,4,5,6], with unsorted array [3,2,6,4,5] we can't perform Binary search.
  2. In Binary search,firstly Find middle value.
  3. Then compare search element with middle value
  4. If middle value is equal to search element then return the index number
  5. else,if the search element is greater than middle value, Ignore the first part of the arry
  6. else ignore the last of the array.
  7. You should perform above process till you find search element, if element not found in the array, then return -1

n = list(map(int,input().split())) //input from the user
n.sort() 
n  =[1,3,6,8,34,45,65,68,76,90,97]

def binarySearch (n,s):
    firstIndex  =0
    lastIndex = len(n)-1
    while (firstIndex <= lastIndex):
        middleIndex = (firstIndex + lastIndex )//2
        if (s > n[middleIndex]):
            firstIndex  = middleIndex +1
        elif (s < n[middleIndex]):
            lastIndex  =middleIndex-1
        else:
            return middleIndex
    else:
        return -1
        
res = binarySearch(n,68)
print(res)
result = 7 (because index value of 68 is 7)
