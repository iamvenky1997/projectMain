Sort the Array

This is the most typical question asked for many freshers to test there basic knowledge. There are many ways to solve this problem. In this article i am going to show you by using bubble sort. In python we can call list as an array. you may use same logic to solve in other programming language. The logic is same for every programm, but syntax may vary.

Steps to solve the problem

  1. In Bubble sort, We can compare adjacent elements.
  2. If interviewer asked for sort the arrar in ascending order. If first element is greater than second one, then you need to swap the elements. Other wise no need to swap.
  3. If he/ she asked for descending order. Vice versa..
  4. You will get better understading by following diagram.


Solution for Sorting Array

Solution for ascending Order :
  • Don't use in-built function In the interview : sort() or sorted()
  • The Interview is all about testing your knowledge and confidence skills.
  • You can Take input from the user by using : n= list(map(int,input().split()))
n  =[4,67,23,1,65,43,78,98,23,2]

def bubbleSort(n):
    while True:
        count = 0
        for i in range(len(n)-1):
            if n[i] > n[i+1]:
                n[i],n[i+1] = n[i+1],n[i]
                count += 1
        if count == 0:
            return n
res = bubbleSort(n)
print(res)
